# LeapStart Examination Portal Prototype

Standalone HTML prototype for the LeapStart School of Technology Examination Portal.

## Extracted from leapstart.in

- Logo assets: `assets/leapstart-logo.webp` and `assets/leapstart-logo-white.webp`
- Organization: LeapStart Ed Tech Private Limited
- Program: Four-year on-campus Computer Science program with specialization in Data Science and Artificial Intelligence
- Positioning: Experiential learning, mentorship, project-first AI and Data Science education
- Contact: info@leapstart.in
- Address: 6th Floor, Sanpra's Corporate Capital, Rd No.2, Hyderabad, India

## Run

Open `index.html` in a browser, or serve the folder locally:

```bash
python3 -m http.server 8080
```
